## Deployment on Ubuntu

### Prerequisites
- Node.js (v14 or higher)
- npm
- PM2 (for process management)

### Steps

1. Install Node.js and npm:
```bash
sudo apt update
sudo apt install nodejs npm
```

2. Install PM2 globally:
```bash
sudo npm install -g pm2
```

3. Clone and setup the project:
```bash
git clone <repository-url>
cd tools
npm install
```

4. Start the application with PM2:
```bash
pm2 start server.js --name "tools-website"
pm2 save
pm2 startup
```

5. Configure Nginx as reverse proxy (optional):
```bash
sudo apt install nginx
```

Create Nginx configuration:
```bash
sudo nano /etc/nginx/sites-available/tools
```

Add the following configuration:
```nginx
server {
    listen 80;
    server_name your-domain.com;

    location / {
        proxy_pass http://localhost:3000;
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection 'upgrade';
        proxy_set_header Host $host;
        proxy_cache_bypass $http_upgrade;
    }
}
```

Enable the site:
```bash
sudo ln -s /etc/nginx/sites-available/tools /etc/nginx/sites-enabled/
sudo nginx -t
sudo systemctl restart nginx
```

6. Configure firewall:
```bash
sudo ufw allow 80
sudo ufw allow 443
sudo ufw enable
```

## Usage

### Keyboard Shortcuts
- `Ctrl/Cmd + Enter`: Execute primary action in current tool

### API Endpoints

All endpoints accept JSON with a `text` field and return JSON with a `result` field.

- `POST /api/url/encode` - URL encode
- `POST /api/url/decode` - URL decode
- `POST /api/base64/encode` - Base64 encode
- `POST /api/base64/decode` - Base64 decode
- `POST /api/unicode/encode` - Unicode encode
- `POST /api/unicode/decode` - Unicode decode
- `POST /api/md5` - Generate MD5 hash
- `POST /api/sha256` - Generate SHA256 hash
- `POST /api/diff` - Compare two texts (accepts `text1` and `text2`)
